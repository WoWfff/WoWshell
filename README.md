# WoWshell

Production-grade modular Quickshell desktop shell for Niri.

## Goals

- maintainability
- performance
- modularity
- responsive architecture
- Material You inspired UI
- runtime stability

## Stack

- Quickshell
- Qt6/QML
- Wayland
- Niri
- Arch Linux

## Design Philosophy

WoWshell focuses on:

- elegant restrained UI
- subtle polished effects
- smooth responsive animations
- lightweight runtime behavior
- maintainable architecture

Animations and visual effects should enhance usability without harming performance.

## Architecture

Recommended structure:

- components/
- widgets/
- modules/
- services/
- state/
- theme/
- utils/
- scripts/
- config/
