# WoWshell Animation Philosophy

## Goals

Animations should:

- enhance usability
- improve perceived smoothness
- feel polished
- remain subtle
- preserve responsiveness

---

## Avoid

- flashy animations
- excessive transitions
- animation spam
- unnecessary GPU-heavy effects
- distracting motion

---

## Preferred Animation Style

Use:

- smooth fades
- subtle scaling
- lightweight transitions
- responsive hover states
- restrained blur effects

Animations should feel modern and elegant without overwhelming the UI.

---

## Performance Rules

Animations must:

- remain performant
- avoid excessive redraws
- avoid expensive reactive updates
- avoid blocking rendering

Always prioritize responsive runtime behavior.
